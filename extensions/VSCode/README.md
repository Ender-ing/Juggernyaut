# Language Support for VSCode

This extension provides language support for Juggernyaut within VS Code.

## Features

- **Syntax Highlighting**: Syntax highlighting for Juggernyaut for an easier experience while coding.

- **Snippets**: Speed up your development with built-in snippets for common Juggernyaut
code.

- **File Associations**: Automatically associate `.jug` files with the Juggernyaut language mode.
